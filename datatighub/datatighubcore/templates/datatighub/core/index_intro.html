What is DataTig?

DataTig is a set of tools to help when a community of people want to crowd source a data set and they use a git repository to store the data.

In these cases, DataTig can be used on the git repository to help people contribute new data or edit existing data, check data quality and transform the data into more useful forms for everyone to re-use.

{{setting_datatig_hub_name}} hosts a number of repositories for everyone to browse. You can also use the tools directly as an open source Python library.
